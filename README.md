# 都道府県別人口推移グラフ生成アプリケーション

このプロジェクトは、ReactおよびNext.jsを使用して都道府県別の人口推移グラフを動的に生成するアプリケーションです。ユーザーは都道府県と人口の種類（総人口、年少人口、生産年齢人口、老年人口）を選択することで、人口推移グラフを確認できます。
